筑波山(9/21)

出發

雖然太陽出來後的溫度跟台灣差不多，但早上的筑波涼快很多，騎車起來很舒服

到了山腳下後情況逐漸不對勁，坡度越來越陡了，圖片可能看不太出來，但感覺應該有20度左右

![20240921_073007](https://hackmd.io/_uploads/HJeYQ7Qlyx.jpg =40%x)
到了一之鳥居 後，坡度直接上升到40度。腳踏車完全上不去，只能用牽的。原本1hr的車程直接變成2hr 😩

![20240921_080235](https://hackmd.io/_uploads/ry5PmXXekx.jpg =40%x)
汽車入口的鳥居，我還是第一次看到這麼大的版本

筑波山神社

經過快一個小時的牽車後，終於抵達了登山口旁，海拔270公尺的筑波山神社拜殿。

:::info
筑波山神社供奉的神即為男體山的伊邪那岐與女體山伊邪那美，至今已有3000年的歷史，參拜的內容通常與結婚與安產有關，不過也會有祈求健康或交通安全等等的參拜。
:::
![20240921_082446](https://hackmd.io/_uploads/BygdQ7mgyx.jpg =40%x)
![20240921_082619](https://hackmd.io/_uploads/Skz_7mQekg.jpg =70%x)
![20240921_082733](https://hackmd.io/_uploads/ByS8Xm7eJl.jpg =70%x)
:::warning
三天前的台灣交換生歡迎會上問了筑波山神社供奉的神體，而我明明去過卻完全不知道，感覺自己對於去過的地方的調查太隨便了，這也是我打算幫旅行做紀錄的原因。
:::
:::info
筑波山位於茨城県つくば市北邊，標高877m，分為西側的男體山與東側的女體山，到了秋天全山楓葉轉紅時赤紅的山體可謂絕景。天氣好時，在筑波山上可以眺望富士山，而在富士山上可以也能看見筑波山，因此有「西富士，東筑波」的美名。

![筑波山 - Google 地圖 - Google Chrome 2024_10_8 上午 08_59_51](https://hackmd.io/_uploads/SyqAm94J1l.png =40%x)
:::
:::success
雖然是日本百名山的一員，但路線親民，此外還有有空中纜車（ロープウェー）與路面纜車（ケーブルカー）可選擇，各路線花費時間如下。

御幸ヶ原コース(登り90分／下り70分)
迎場コース(登り40分／下り35分)
白雲橋コース(登り110分／下り95分)
おたつ石コース(登り40分／下り35分)
自然研究路(60分)
山頂連絡路(15分)

![normal](https://hackmd.io/_uploads/ByKIyZGy1l.gif =80%x)
:::

因為這次是第一次爬，而且前面還騎了快2hr，所以保守點選了 御幸ヶ原コース=>山頂連絡路=>白雲橋コース 的路線。

御幸ヶ原コース

整體路線並不陡，而且1.5hr內就可以到山頂，但前面騎+牽已經精疲力盡了，好想回家…
![20240921_093249](https://hackmd.io/_uploads/HyrXf4Qlyx.jpg =40%x)
![20240921_093545](https://hackmd.io/_uploads/rJdmMVmekx.jpg =40%x)
![20240921_092659](https://hackmd.io/_uploads/SyxH777x1l.jpg =40%x)

![20240921_091913](https://hackmd.io/_uploads/HkrvX7Xeyg.jpg =40%x)
御幸ヶ原コース一側的是路面纜車的軌道，如果想搭乘空中纜車的話需要從女体山駅或つつじヶ丘駅搭乘。

抵達纜車站終點

![20240921_105709](https://hackmd.io/_uploads/SJxmHm7mlyl.jpg =80%x)
山頂上大概有四到五間的餐廳，價格只比一般餐廳高一點，目前在日本的觀光區看來，像台灣那樣的觀光區盤子店真的少之又少

男體山頂

![20240921_103905](https://hackmd.io/_uploads/H1mVXm7e1e.jpg =70%x)

![20240921_103355](https://hackmd.io/_uploads/BkIE7Q7x1e.jpg =90%x)
雲霧有點多，所以沒辦法看得太遠，聽說天氣好時可以看到富士山

女體山頂

![20240921_111838](https://hackmd.io/_uploads/HykEmQQxye.jpg =80%x)
這邊的霧比男體山上濃太多了，根本看不見

山頂連絡路 + 白雲橋コース

若是想坐兩種纜車上下山，就必須走過山頂連絡路，也因此這條也是最熱鬧的路線，看點則是各式各樣形狀的石頭。

在神社時有遇到一群應該是社區活動的登山團，下山路上剛好遇到他們再發多出來的冰淇淋。

下山的路線更為平坦，但路線也更長

總結

去程2hr + 登山5hr + 回程1hr，不過登山的路線非常輕鬆，除了那段陡坡，騎在田園間也很舒服，若覺得騎車太累的也可以搭巴士至登山口，大推 :+1: (而且這附近的觀光景點也就這裡而已了

可惜這次去楓葉還沒有轉紅，一兩個月後可能會為了楓葉再去一趟，不過應該是搭公車啦，那段牽車真的太可怕了
